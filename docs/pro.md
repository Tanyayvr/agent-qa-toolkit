# Pro / Pro+ (Paid, Optional)

The core Agent QA Toolkit is open-source and self-hosted.
Paid tiers are optional and focus on packaging/support, not lock-in.

## OSS vs Paid Boundary

| Area | OSS (in this repo) | Pro / Pro+ (paid) |
| --- | --- | --- |
| Runner/evaluator pipeline | Included | Included + support SLA |
| Portable reports (`report.html`, `compare-report.json`, `manifest.json`) | Included | Included + onboarding and rollout support |
| Security scanners | Included | Included + curated scanner rules updates (Pro roadmap) |
| Historical Trending engine (`npm run trend`, SQLite, trend.html) | Included | Governed retention operations, migration support, and enterprise rollout support |
| Integrations (adapters/plugins) | Included reference implementations | Integration hardening support for production stacks |

## Pro (roadmap)
- Curated scanner rules library and managed update cadence.

## Pro+ (current commercial packaging)
- Historical trending operations support (retention, rollout, policy guardrails).
- Priority support for release-gate tuning and incident forensics workflows.

## Pilot (free for early teams)
We offer free pilot access to Pro / Pro+ packaging/support for a limited number of teams.

Apply via `docs/pilot.md` or the pilot issue template.
